# Simplified Posture AI

A streamlined posture detection system using MediaPipe and FastAPI.

## 🚀 Quick Start

1. **Install Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the Server:**
   ```bash
   python app.py
   ```

3. **Access the Dashboard:**
   Open `http://localhost:8080` in your browser.

## 🛠️ Features
- **Live Stream:** Real-time video feed with pose overlay.
- **Telemetry:** Live tracking of Pitch, Yaw, and Roll.
- **Calibration:** One-click baseline setting.
- **Background Removal:** Automatic background segmentation for privacy and focus.
